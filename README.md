# MuseGeo
Suivi NDVI pour parcelles agricoles

MuseGeo est une démo de suivi NDVI pour parcelles agricoles. Une interface cartographique web légère, 100% front-end et locale (LocalStorage), conçue pour sonder l'intérêt des éleveurs sur le terrain. Filtre les nuages, simule des courbes de biomasse réalistes et intègre un module de feedback utilisateur pour valider le besoin AgTech avant de coder la plomberie cloud (Copernicus/Supabase).
